## Cool Features

* You need to shoot on beat and get stunned if you dont
* You can pick a skin
* Supports both azerty and qwerty
* Game gets more difficult over time
* Has a responsive GUI:
* You can pick up powerups
* There is a secret easter egg
* Cross platform
  * Windows, macOS and linux (gnome).

## How To Play

You need to shoot on the beat. Time youre shots and use powerups and health packs to stay alive

> **Note**
> If you're using Linux Hyperland or a other tilling wm, Youre bar might still be on top once game loads. Manually setting fullscreen resolves this
